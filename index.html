Google Maps

    Google Maps es un servicio de mapas para la Web desarrollado por Google.

    Incluye imágenes de satélite, mapas de carretera, vistas de 360°, condiciones de tráfico en tiempo real y planificación de rutas.

    Es uno de los productos de mayor influencia en el área de los sistemas de información geográfica.

Historia de Google Maps

    Fue creada, como una aplicación de escritorio, por los daneses Lars y Eilstrup Rasmussen en Where 2 Technologies.

    En 2004, la compañía fue adquirida por Google, que la transformó en una aplicación web.

    Fue lanzada en febrero de 2005.

El API de Google Maps

    Google Maps cuenta con un Application Programming Interface (API) propietaria que permite acceder a sus datos e incrustar su funcionalidad en otras aplicaciones.

    El API de Google Maps puede accederse desde JavaScript y otros lenguajes (ej. Python, Java).

Programación de un mapa de Google Maps

  1. Crear un documento HTML
  2. Cargar el API
  3. Crear un contenedor para el mapa
  4. Especificar las opciones del mapa
  5. Crear el objeto mapa
  6. Cargar el mapa

1. Crear un documento HTML


                        <!DOCTYPE html>
                        <html lang="es">
                            <head>
                                <title></title>
                                <meta charset="UTF-8"> 
                            </head>
                            <body>
                            </body>
                        </html>

2. Cargar el API

    El API se carga al incluir su dirección en el tag <script>


                        <!DOCTYPE html>
                        <html lang="es">
                            <head>
                                <title></title>
                                <meta charset="UTF-8"> 
                                <script src = "https://maps.googleapis.com/maps/api/js"> 
                            </head>
                            <body>
                            </body>
                        </html>

3. Crear un contenedor para el mapa

    El contenedor es el elemento HTML que contendrá el mapa. Generalmente, se usa el tag genérico <div>


                        <div id="mimapa" style="width:900px; height:580px;"></div>

4. Especificar las opciones del mapa

    Junto con el mapa, debe crear un objeto MapOptions que especifica el centro del mapa, el nivel de acercamiento y el tipo de mapa, entre otras. Es recomendable hacer esto con una función.


                        function cargarMapa() {
                           var opcionesMapa = {
                              center:new google.maps.LatLng(10, -84), 
                              zoom:7, 
                              mapTypeId:google.maps.MapTypeId.ROADMAP
                           };
                        }

5. Crear el objeto mapa

    El objeto mapa se crea al instanciar la clase Map. Deben especificarse el contenedor y las opciones del mapa.


                        var mapa = new google.maps.Map(document.getElementById("mimapa"), opcionesMapa);

6. Cargar el mapa

    Finalmente, el mapa se carga (por ejemplo) en el evento onload.


                        <body onload = "cargarMapa()">

Ejemplo completo


                        <!DOCTYPE html>
                        <html lang="es">  
                           <head>
                              <title>Mi primer mapa</title>
                              <script src = "https://maps.googleapis.com/maps/api/js"></script>
                              <meta charset="UTF-8">
                              <script>
                                 function cargarMapa() {
                                    var opcionesMapa = {
                                       center:new google.maps.LatLng(10, -84), 
                                       zoom:6, 
                                       mapTypeId:google.maps.MapTypeId.ROADMAP
                                    };
                                    var mapa = new google.maps.Map(document.getElementById("mimapa"),opcionesMapa);
                                 }
                              </script>
                           </head>
                           <body onload = "cargarMapa()">
                              <div id="mimapa" style="width:580px; height:400px;"></div>
                           </body>
                        </html>
